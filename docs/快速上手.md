# ⚡ V2EX Max Helper — 快速上手

> 以下是你从零到运行的 3 个步骤。

---

## 🪜 步骤 1：获取 V2EX Cookie（必须）

1. 浏览器打开 https://www.v2ex.com 并**登录**
2. F12 → Application → Cookies → `www.v2ex.com`
3. 复制完整的 Cookie 值（长字符串）

```
示例格式:
A2=xxxxx...; V2EX_TAB=...; ...
```

## 🪜 步骤 2：写入 Cookie 到服务器

SSH 到 `8.135.36.248`：

```bash
# 替换 <你的Cookie> 为实际值
nano /root/.v2ex_cookie
# 粘贴你的 Cookie，保存退出 (Ctrl+O, Ctrl+X)

# 验证
cat /root/.v2ex_cookie
```

## 🪜 步骤 3：测试运行

```bash
# 测试签到（代理仅对 v2ex 进程生效，不影响服务器其他业务）
cd ~/v2ex-max-helper/checkin && node --require ./preload-proxy.js v2ex-checkin.js

# 测试阅读（5 篇限制）
cd ~/v2ex-max-helper/reader && node --require ./preload-proxy.js main.js --limit 5
```

如果签到成功，说明一切 OK！系统会自动按定时器运行。

---

## 📱 飞书通知（已配置）

飞书 Webhook 已配置到 `/root/.v2ex_env`，签到和阅读结果会自动推送到你的飞书群。

```bash
# 查看配置
cat /root/.v2ex_env
```

飞书机器人安全设置：「自定义关键词」填 `V2EX` 即可。

---

## 🔐 Cookie 会自动续期，无需担心失效

项目已内置保活机制：
- **每 6 小时** 自动访问 V2EX 首页 → 触发滑动续期
- **自动捕获新 Cookie** → 写回 `/root/.v2ex_cookie`
- **Cookie 失效时** → 飞书立刻通知你

> 正常情况下，你写入一次 Cookie 后就不需要再管了。
